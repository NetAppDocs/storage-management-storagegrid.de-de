= Anzeigen und Aktualisieren von StorageGRID -Zertifikaten in der NetApp Konsole
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Auf Ihrem StorageGRID -System ist ein Management-Schnittstellenzertifikat erforderlich, bevor Sie das System über die NetApp -Konsole erkennen.  Das Zertifikat stellt eine sichere Verbindung zwischen der Konsole und StorageGRID her.  Da Zertifikate ablaufen können, müssen Sie gelegentlich möglicherweise den Status anzeigen oder ein Zertifikat aktualisieren.

.Erforderliche NetApp Konsolenrolle:
Speicheradministrator.link:https://docs.netapp.com/us-en/bluexp-setup-admin/reference-iam-predefined-roles.html["Erfahren Sie mehr über die Zugriffsrollen der NetApp Konsole."^]



== Den Status eines Management-Schnittstellenzertifikats anzeigen

Zeigen Sie den Status des Verwaltungsschnittstellenzertifikats für ein StorageGRID -System an, um zu überprüfen, ob das Zertifikat noch gültig ist.

.Schritte
. Wählen Sie im Navigationsmenü *Speicher > Verwaltung*.
. Wählen Sie auf der Seite *Systeme* das StorageGRID -System aus.
. Überprüfen Sie in der Standardansicht den Status des vom Benutzer genehmigten Zertifikats unten im Bereich „Informationen“ auf der rechten Seite der Seite.
+

NOTE: Der Zertifikatsstatus wird alle 24 Stunden aktualisiert.





== Aktualisieren des Management-Schnittstellenzertifikats

Wenn der Status Ihres Management-Schnittstellenzertifikats ungültig ist oder Sie eine Benachrichtigung erhalten, dass Ihr Zertifikat abgelaufen ist, müssen Sie es aktualisieren, damit die NetApp Konsole eine Verbindung zu StorageGRID herstellen kann.

.Schritte
. Gehen Sie zu Ihrem lokalen Grid Manager in StorageGRID und folgen Sie den Anweisungen für https://docs.netapp.com/us-en/storagegrid-118/admin/configuring-custom-server-certificate-for-grid-manager-tenant-manager.html#add-a-custom-management-interface-certificate["Installieren eines Management-Schnittstellenzertifikats auf StorageGRID"] .
. Wählen Sie im Navigationsmenü der NetApp -Konsole *Storage > Management* aus.
. Wählen Sie auf der Seite *Systeme* das StorageGRID -System aus.
. Wählen Sie das Symbol „Weitere Optionen“ (Kreis mit drei vertikalen Punkten).
+
image:screenshot-update-certificate.png["Ein Screenshot des Update-Menüs."]

. Wählen Sie *Zertifikat aktualisieren*.
. Wählen Sie auf der Seite „Zertifikat genehmigen“ die Option „Zertifikat genehmigen“ aus, um das aktualisierte Zertifikat zu verwenden.
